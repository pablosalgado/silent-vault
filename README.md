# Silent Vault

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

Captures all notifications, silences them, stores in a local database, and shows a simple UI to review unreviewed notifications.

## Stack

Kotlin · Jetpack Compose · Room · NotificationListenerService

## Prerequisites

- **GitHub CLI**: authenticated and configured.

```sh
brew install gh      # macOS
gh auth login        # then follow the prompts
```

## Build

```sh
./gradlew assembleDebug
```
